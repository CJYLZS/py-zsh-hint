## py-zsh-hint.plugin.zsh

### keybind:

- `ctrl + u` : start

- `alt + o` : selected history forward

- `alt + b` : selected history back

- `alt + h` : left option 

- `alt + l` : right option

- `alt + j` : down option

- `alt + k` : up option

- `alt + p` : parent dir

- `tab` : chose option

- `enter` : run cmd

- `ctrl + c` : cancel

### install

```shell
# used in zsh
source /path/to/py-zsh-hint/py-zsh-hint.plugin.zsh
```
